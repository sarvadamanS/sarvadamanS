## 🚀 Featured Projects

### 🪙 Aurum — Gold Price Tracker
> **Next.js · Supabase · Vercel Cron · Resend**

A personal gold price intelligence dashboard that fetches live prices daily, stores historical data, and sends email alerts on significant price drops.

- Automated weekday price fetching via Vercel Cron Jobs (12 PM IST)
- Stores historical data in Supabase PostgreSQL — builds your own dataset over time
- Email alerts via Resend when gold drops >5% below 7-day average
- Beautiful animated UI with parallax hero, gold orb, and area charts
- Displays prices in USD and INR (per troy oz + per 10g)
